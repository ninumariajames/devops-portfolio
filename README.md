## 🐳 Step 2: Running My App with Docker

### ✅ What I Did

In this step, I took my Flask application and ran it inside a Docker container.

* Created a Dockerfile for my project
* Built a Docker image from my code
* Ran the application inside a container

---

### ▶️ Commands I Used

To build the Docker image:
docker build -t myapp .

To run the container:
docker run -p 5000:5000 myapp

---

### 📚 What I Learned

* How Docker helps package an application with all its dependencies
* How to create and run containers
* Basic understanding of containerization in DevOps

---

### 💡 My Experience

This was my first time using Docker. Initially, I faced some errors, but after fixing them, I was able to successfully run my app inside a container. This step helped me understand how applications can run consistently in any environment.


## ☁️ Step 3: Deploying My App on AWS

### ✅ What I Did

In this step, I deployed my application to the cloud using AWS.

* Created a virtual server (EC2 instance) on AWS
* Connected to the server using SSH
* Installed Docker on the server
* Uploaded my project files from my local system
* Built and ran the Docker container on the server

---

After completing all the steps, I was able to access my application through the public IP address in a browser. Seeing my project live on the internet was a great moment for me.
